# Vision: MentalMath

A web app that makes mental math training addictive and measurable.

## What This Is

A fullstack mental math trainer — timed drills, adaptive difficulty, progress tracking, and competitive leaderboards. Built to help people get faster at arithmetic in their head, from basic addition to multi-digit multiplication and beyond.

## Who It's For

Students, engineers, traders, competitive math enthusiasts — anyone who wants to sharpen their mental calculation speed and accuracy.

## What Makes Someone Say "This Is Amazing"

1. You open the app and you're solving problems in under 3 seconds. No signup friction, just start.
2. The difficulty adapts to YOU — too easy and it ramps up, struggling and it backs off. Always in the sweet spot.
3. You can see your speed and accuracy improve over time with charts and streaks.
4. Compete with friends or the global leaderboard. Daily challenges keep you coming back.

## Roadmap

### Phase 1 — Core Training Engine

#### Quick Wins
- [ ] Problem generator — addition, subtraction, multiplication, division with configurable difficulty
- [ ] Timed drill mode — solve as many as you can in 60 seconds, track score
- [ ] Answer input with instant feedback — green/red flash, show correct answer on miss
- [ ] Basic stats dashboard — problems solved, accuracy %, average response time
- [ ] Dark mode UI with a clean, focused design — no distractions during drills

#### Big Bets
- [ ] Adaptive difficulty engine — tracks per-operation accuracy, adjusts number ranges automatically
- [ ] Progress tracking with charts — daily/weekly/monthly trends for speed and accuracy
- [ ] User profiles with auth — save progress across devices, personal bests

#### Moonshots
- [ ] Daily challenge mode — same problem set for everyone, global leaderboard
- [ ] Multiplayer head-to-head — real-time races against other players via WebSocket
